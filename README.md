# Documentation
## File `.env`
This file must be in repository
```
# optional
LOG_LEVEL= # 1 - Fatal Error, 2 - Error, 3 Warning, 4 - Info, 5 - Debug, 3 by default
# required
PORT= # server port

# required
MUSIC_INFO_URL= # url to get data of new song, example: https://google.com

# required (all)
DB_USER= 
DB_PASS= 
DB_NAME=
DB_PORT=
DB_HOST=
```
